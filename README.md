# 🛍️ Frontend Store

Tienda en línea de camisetas temáticas para desarrolladores. Incluye catálogo de productos, página de detalle, formulario de compra y sección "Nosotros". Desarrollada con HTML5 y CSS3 puro, con diseño responsivo usando CSS Grid.

🔗 **[Ver Demo en vivo](https://noahxd-dev.github.io/Frontrnd-Store/)**

---

## ✨ Características

- **Catálogo de productos** con 14 camisetas temáticas de tecnologías (Vue, React, Node, etc.)
- **Página de detalle** con selector de talla, cantidad y botón de agregar al carrito
- **Sección Nosotros** con información del negocio y bloques de beneficios
- **CSS Grid responsivo** adaptado para móvil (2 columnas) y escritorio (3 columnas)
- **Gráficos decorativos** integrados en el grid del catálogo
- **Navegación** con estado activo por página
- **Año dinámico** en el footer generado con JavaScript
- **Tipografía personalizada** con Google Fonts (Staatliches)
- **Variables CSS** para una paleta de colores consistente
